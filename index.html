📖 可用命令：
plant("wheat") - 种植小麦 (成本: 0 | 收益: +10)
plant("corn") - 种植玉米 (成本: 50 | 收益: +80 | 净赚: +30)
plant("gold") - 种植金麦 (成本: 150 | 收益: +300 | 净赚: +150)
harvest() - 收割作物
sell() - 出售作物
checkMoney() - 查看金币
checkState() - 查看农田状态
canAfford("corn") - 检查是否可以买玉米
wait(3) - 等待3秒（返回 Promise，需配合 await 使用）
checkLoopLimit() - 检查循环限制（在循环中使用）
💡 示例：使用 if/else
if (money >= 50) { plant("corn"); } else { plant("wheat"); }
🔄 示例：使用 while 循环 + async/await
// 使用 async/await 和 wait() 函数（推荐） (async function() { while (money < 500) { checkLoopLimit(); if (state === "empty") { if (money >= 150) { plant("gold"); await wait(8); // 等待8秒，不阻塞浏览器 } else if (money >= 50) { plant("corn"); await wait(5); // 等待5秒 } else { plant("wheat"); await wait(3); // 等待3秒 } } else if (state === "ready") { harvest(); } else if (state === "harvested") { sell(); } else if (state === "growing") { await wait(1); // 等待1秒后继续检查 } } })(); // 简单版本：只种小麦 (async function() { while (money < 1000) { checkLoopLimit(); if (state === "empty") { plant("wheat"); await wait(3); } else if (state === "ready") { harvest(); } else if (state === "harvested") { sell(); } else if (state === "growing") { await wait(1); } } })();
💡 提示：按 Ctrl+Enter 执行代码
⚡ 推荐：使用 async/await 和 wait() 函数，不会卡死浏览器
⚠️ 注意：代码最多执行30秒，防止无限循环
⚠️ 重要：使用 await wait() 时，代码必须在 async 函数中
